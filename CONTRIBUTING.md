# Contributing to JINGI Artifacts

JINGIでは、コードだけでなく  
**検証・再現・整理・設計・考察** も重要な貢献として扱います。

---

## 1. 成果提出の流れ（概要）

1. テンプレートからリポジトリを作成
2. 成果を整理・記録
3. `jingI-proof.txt` による本人証明
4. JINGIアプリから登録
5. 再現・検証が進むと評価が強化されます

---

## 2. 推奨される貢献の種類

- 実装（モデル・推論・評価・UI）
- 再現実験・比較検証
- データ作成・品質検証
- ドキュメント整備
- ベンチマーク設計

---

## 3. 再現報告について

- Issueテンプレート「Reproduction Report」を使用してください
- 再現成功・失敗の両方が価値ある貢献です

---

## 4. 行動規範

- 他者の成果・時間・努力を尊重してください
- 不明点はIssueで質問してください
- 攻撃的・排他的な行為は禁止です
