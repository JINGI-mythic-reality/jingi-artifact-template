# JINGI Artifact Repository

このリポジトリは、JINGIプロジェクトにおける **学習・研究・制作の成果（Artifact）** を記録・共有するためのものです。

本成果は、ゲームアプリ外で行われた創作活動（実験・検証・実装・整理）を  
**世界樹のノード（外部成果）** としてゲーム内に反映することを目的としています。

---

## Artifact概要

- Artifact名：
- 種類（tag）：
  - artifact:dataset / benchmark / training / inference-opt / rag / app-ui など
- 神器性（tag）：
  - sanki:ken / sanki:kagami / sanki:magatama

- 想定される利用者：
